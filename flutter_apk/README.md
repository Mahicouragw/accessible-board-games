# Accessible Board Games — Android App (Flutter wrapper)

This folder is a minimal Flutter app that wraps the live web app
(https://accessible-board-games.vercel.app) in a native WebView.

- 19 games, realistic sounds and per-game music update instantly from the web.
- TalkBack-friendly, back-aware navigation, offline error screen with retry.

The APK is built automatically by GitHub Actions (`.github/workflows/apk.yml`)
and published under **Releases → flutter-apk-N**.

App ID: `com.mahicouragw.board_games`
